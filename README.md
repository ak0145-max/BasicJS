This is my very first JavaScript practice project. I built it while learning web development. I'm really excited to share it!

## What I Made
I created 4 small but useful mini projects:
1. **Number Guessing Game**  
   The computer picks a secret number between 1 to 100. You have to guess it. It tells you if your guess is too high or too low and counts your attempts. There's also a "New Game" button to play again.

2. **Length Unit Converter**  
   You enter a number and choose a unit (km, m, cm, or mm), and it automatically shows the value in all other units. Super handy!

3. **Bangladesh Digital Clock**  
   A live clock that shows current Dhaka time in 12-hour format with AM/PM. It also changes the greeting depending on the time of day (Good Morning, Good Evening, etc.).

4. **Smart Form Validator**  
   A simple form where I added validation for name, email, age, and terms checkbox. It shows error messages if something is wrong and a success message when everything is correct.
## What I Learned
- How to make websites interactive using JavaScript  
- Working with buttons, inputs, and real-time updates  
- Handling time and timezones (especially Bangladesh time!)  
- Basic form validation  
- Writing cleaner and commented code

This project helped me understand DOM manipulation, event listeners, and conditional logic much better.
